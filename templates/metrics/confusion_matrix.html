Performance Analysis

This matrix evaluates the model on 1,737 unseen test samples. The diagonal values (dark blue) represent correct predictions, while the off-diagonal values represent errors.

  • True Negatives (846): Healthy subjects correctly identified as Healthy.
  • True Positives (861): Depressed subjects correctly identified as Depressed.
  • False Positives (10): Healthy subjects incorrectly flagged as Depressed (False Alarm).
  • False Negatives (20): Depressed subjects incorrectly missed and labeled as Healthy.

Conclusion: The model achieved an overall accuracy of 98.27%, with a very low error rate (only 30 errors out of 1,737 cases), demonstrating high reliability for screening purposes.
